- 最新Xアルゴリズムを踏まえた実践テクニック

## 11-1. ビューとは

### 定義
**ビュー（View）= あなたの投稿が誰かの画面に表示された回数**です。

```
ビュー = チラシを配った枚数
    ↓
チラシを受け取った人（ビュー）の中から
    ↓
内容を読んでくれた人（エンゲージメント）が生まれ
    ↓
お店に来てくれる人（プロフィール訪問）が生まれる
```

> つまり、ビューが取れないということは、
> チラシを誰にも配れていない状態と同じです。

### ビューは全ての起点

```
ビュー（見てもらう）
    ↓
エンゲージメント（反応してもらう）
    ↓
プロフィール訪問（知ってもらう）
    ↓
リンクタップ（行動してもらう）
    ↓
報酬
```

ビューが取れなければ、それ以降の全てが発生しません。ジャンル認知もビュー獲得がベースです。

## 11-2. ビューを獲得するための具体的な行動

### ①投稿でビューを取る

| 施策 | 具体的な行動 |
|---|---|
| 投稿頻度を上げる | 1日3〜5投稿を確実に出す |
| 投稿時間を最適化 | ターゲットがXを見ている時間に投稿する |
| ハッシュタグの活用 | トレンドハッシュタグ、ジャンル関連タグを使う |
| 画像・動画の活用 | テキストのみより視覚的要素がある方がビューを取りやすい |
| 冒頭の1行で惹きつける | スクロールを止めてもらう最初の1行にこだわる |

### ②リプでビューを取る

| 施策 | 具体的な行動 |
|---|---|
| ビューが多い投稿にリプする | ビュー1,000以上の投稿にリプを送る |
| 新しい投稿を狙う | 投稿されてすぐの投稿にリプすると上位表示されやすい |
| リプの質を上げる | 上位表示されるリプ = ビューが取れるリプ |

### ③引用リポストでビューを取る
- 影響力のあるアカウントの投稿を引用リポスト
- 自分の言葉で価値を追加
- 元投稿者がRTしてくれれば爆発的なビューが発生

### 避けるべきこと
- 外部リンクを直接投稿に貼る（ビューが下がりやすい）
- 同じ文面の繰り返し投稿
- 意味のない短文投稿

## 11-4. インプレッションとは

### ビューとの違い

| 指標 | 範囲 | 説明 |
|---|---|---|
| ビュー | 1投稿単位 | その投稿が何回表示されたか |
| **インプレッション** | アカウント全体 | あなたの全投稿が合計何回表示されたか |

```
ビュー = 1枚のチラシが見られた回数
インプレッション = あなたが配った全チラシが見られた合計回数
```

> 週間インプレッション50,000以上あるアカウントは、
> 初期のフォロー対象として推奨されるほどの影響力を持っています。

---

## 11-5. インプレッション向上の施策

### ①投稿の量を増やす

| 方法 | 効果 |
|---|---|
| 1日の投稿数を増やす | 表示のチャンスが増える |
| バイラルダッシュボードの活用 | 効率的に投稿数を確保 |
| 予約投稿の活用 | 最適な時間帯に自動投稿 |

### ②投稿のタイミングを最適化

| 時間帯 | 特徴 |
|---|---|
| 朝6:00〜8:00 | 通勤時間帯、多くのユーザーがXをチェック |
| 昼12:00〜13:00 | ランチタイム、短時間にビューが集中 |
| 夜20:00〜22:00 | 帰宅後、SNS利用のピーク時間 |

### ③インタラクションを増やす

| 方法 | 効果 |
|---|---|
| リプの数を増やす | あなたのリプ自体もインプレッションに含まれる |
| 引用リポスト | 拡散によるインプレッション大幅増 |
| 相互交流 | お互いの投稿にインプレッションが加算される |

### 確認方法
1. Xの「アナリティクス」ページにアクセス
2. 週間・月間のインプレッション数を確認
3. どの投稿がインプレッションを稼いでいるか分析
4. 毎週のインプレッション数をFBフォームに記入

---

## まとめ

- **ビュー = 1投稿の表示回数**、全ての起点。ビューなくして報酬なし
- **インプレッション = アカウント全体の表示回数**、影響力の指標
- 投稿・リプ・引用リポストの3つでビューを獲得する
- 投稿の量×質×タイミング×インタラクションの掛け算でインプレッションが決まる
- 毎週の数字を記録し、右肩上がりを目指す
