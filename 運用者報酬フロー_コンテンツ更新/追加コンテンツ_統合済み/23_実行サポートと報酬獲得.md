# 第23章：実行サポート ＆ 報酬獲得 ── 実行フォローから初報酬とその先へ

---

## この章で学ぶこと

- 実行前後での運用者としての関わり方
- パーソナライズに基づく適切な立ち位置
- 初報酬獲得の意味と価値
- ポートフォリオとしての活用方法
- 報酬獲得後のスケール戦略

---

## 23-1. 実行前後の位置づけ

### 全体の流れの中での位置

```
見込み顧客獲得 → 商談 → 【実行前後】 → 報酬獲得
```

実行前後は、見込み顧客が実際にアクション（物販の購入・参加等）を起こす段階です。

---

## 23-2. 実行前の関わり方

### パーソナライズとの整合性チェック

| 自分の状態 | 適切な関わり方 |
|---|---|
| 自分も物販を経験済み | 体験者として具体的なアドバイス |
| 自分はまだ未経験 | マインドセット面のサポートに集中 |
| 自分のジャンルと合致 | 積極的にサポート |
| 自分のジャンルと不一致 | 運営に引き継ぎ |

> 自身のパーソナライズされたアカウントと相談をし、
> 見つめ直した上で関わりを持つのが良い。

---

## 23-3. 実行後の関わり方

### 主な役割
**マインドセット部分の教育ができるのが良い。**

- 「大丈夫ですよ」と安心感を与える
- 自分の経験（または知っている事例）を共有する
- 次のステップを一緒に確認する
- 過度にフォローし過ぎない

### 運営との連携
- 自分で判断できない場合は運営側に相談する
- 見込み顧客には「確認して連絡します」と伝える
- チームとして見込み顧客をサポートする意識

---

## 23-4. 初報酬獲得の意味

### 単なる収入ではない

初報酬獲得は、あなたのSNS運用のフロー全体が**正しく機能した証拠**です。

```
アカウント作成 → 投稿 → リプ → DM → 信頼構築
→ プロフィール訪問 → リンクタップ → 公式LINE追加
→ 見込み顧客獲得 → 商談 → 実行 → 報酬獲得 ✅
```

> 一連のフローが1回でも回ったという経験は、
> 何よりもの自信になり、以後の運用を加速させます。

---

## 23-5. ポートフォリオとしての活用

### なぜポートフォリオが重要か

報酬を獲得したという事実は、あなたのアカウントに**権威性**を与えます。
- 「実際に稼いでいる人」としての説得力
- ターゲットへの強力なアピール
- フォロワーの信頼感の向上

### 共有する際のルール

> 報酬を共有する際は、**運営側に一言かけてください。**
> 具体的に共有可能な部分と不可能な部分を明記したマニュアルを案内します。

### 共有の方法
- 画像での共有（報酬画面のスクリーンショット等）
- テキストでの共有（体験談・ストーリー）
- 投稿としての共有（エンゲージメントを生む実績投稿）

```
報酬獲得 → ポートフォリオとして公開
    ↓
「この人は実際に稼いでいる」という信頼
    ↓
さらなるフォロワー増加 → リンクタップ増加
    ↓
さらなる報酬獲得 → 好循環の完成
```

---

## 23-6. 報酬獲得後のスケール

### ① 2アカウント目の運用

**1件の報酬獲得ができることによって大枠の流れが掴めます。**

- **X（旧Twitter）の2つ目のアカウント**の立ち上げ
- 異なるペルソナ or 異なるターゲットでの運用
- 1つ目の経験を活かした高速立ち上げ

### ② マルチプラットフォーム展開

| プラットフォーム | 活用方法 |
|---|---|
| **Threads** | Xの投稿を横展開、テキスト中心の発信 |
| **YouTube** | 教育コンテンツ、体験談動画の発信 |
| **Instagram** | ビジュアルでのブランディング強化 |

### ③ CCインフラの活用
報酬がある程度安定してきたら、物販以外のCCインフラの活用を検討します。

## まとめ

- 実行前後は**パーソナライズとの整合性**を考えて関わる
- 経験者として振る舞えるなら積極的に、そうでなければマインドセットサポートに集中
- **初報酬 = フロー全体が機能した証拠**
- ポートフォリオとして活用し、さらなる集客・報酬獲得につなげる
- 報酬獲得後は2アカウント目・マルチプラットフォーム展開を視野に入れる
- **1件の成功体験が、全てを加速させる**
