## 16-1. いいねとは

### 定義
**いいね = 相手の投稿に対する最もシンプルな肯定のアクション**です。

ハートマーク（♥）を押すだけの行動ですが、SNS運用においては**非常に重要な戦略ツール**です。

### いいねが持つ3つの戦略的価値

#### ①自分のアクティブ度を示す
- いいねを多く押しているアカウント ＝ アクティブに利用しているアカウント
- アルゴリズムが「このアカウントはアクティブだ」と判断する

#### ②相手への存在アピール
- いいねを押すと相手に通知が届く
- 何度もいいねを押していると認知される
- 認知 → プロフィール訪問 → フォロー、のきっかけに

#### ③インタラクションの蓄積
- 単体のエンゲージメント重みは低い（★★☆☆☆）
- しかし**継続的な蓄積**で関係性の構築に貢献
- リプやDMへの橋渡しとなるアクション

### 毎日のいいね目標

| 段階 | 目標数/日 |
|---|---|
| 初期 | 30〜50回 |
| 中期 | 50〜100回 |
| 長期 | 状況に応じて |

### いいね × リプのコンビネーション

```
① 相手の投稿にいいねを押す
② さらにリプを送る
③ 相手がリプを返してくれたらいいねを押す
④ 継続的に交流が生まれる
```

### いいねの注意点
- ❌ 短時間に大量のいいね（スパム判定のリスク）
- ❌ 読まずに機械的にいいねを押す
- ⭕ 1件1件、投稿を見てからいいねを押す
- ⭕ 間隔を空けて自然なペースで

## 16-2. 引用リポストとは

### 用語の整理

| 用語 | 説明 |
|---|---|
| RT（リツイート） | 相手の投稿をそのまま共有する |
| 引用リポスト | 相手の投稿に**自分の言葉を添えて**共有する |

### なぜ引用リポストが最強なのか

| 比較項目 | 通常RT | 引用リポスト |
|---|---|---|
| 自分の言葉 | 入らない | **入れられる** |
| 元投稿者からのRT | 起こりにくい | **起こりやすい** |
| 拡散の爆発力 | 限定的 | **爆発的** |
| ジャンル認知 | 弱い | **強い** |

### 爆発的拡散のメカニズム

```
あなたが引用リポスト
    ↓
元の投稿者に通知が届く
    ↓
元の投稿者がRTしてくれる
    ↓
元の投稿者のフォロワーに拡散
    ↓
プロフィール訪問 → リンクタップ → 報酬
```

> **通常のリツイートよりも引用リポストを優先する。**

## 16-3. 効果的な引用リポストの書き方

### 基本構成

```
①共感・賛同（元の投稿への反応）
②自分なりの価値追加（体験・知識・補足）
③メッセージ or CTA
```

### 実践例

#### 例1：共感 ＋ 体験談
```
まさにその通りだと思います。

自分も最初は「いつか始めよう」と3ヶ月先延ばしにしてた。
でもある日「今日やらなかったら一生やらない」と思って
小さく始めたのがターニングポイントでした。

0→1が一番大変。でも一番の成長ポイントでもある。
```

#### 例2：補足情報 ＋ 価値提供
```
この3つ、SNS始めたての頃まんまと全部当てはまってた😂

追加するなら「反応がないからとすぐに方向転換してしまう」も。

最低1ヶ月は同じ方向で運用してみないと、
アルゴリズムにジャンル認知されない。

継続力、マジで大事です。
```

### 引用リポストの目標

| 段階 | 1日の目標 |
|---|---|
| 初期（0〜50） | 1〜2件 |
| 中期（50〜500） | 2〜3件 |
| 長期（500〜） | 3〜5件 |

### 注意点
- ❌ 元の投稿を否定するような引用はしない
- ❌ 自分の言葉を入れない（ただのRTと同じ）
- ⭕ 多様な人の投稿を引用して交流の幅を広げる
- ⭕ 元の投稿者へのリスペクトを忘れない

## まとめ

- **いいねはSNS運用の「基礎呼吸」**、積み重ねが大きな効果を生む
- **引用リポスト = 拡散力最大の武器**
- いいね→リプ→DM のステップアップに活用する
- 引用リポストは自分の言葉を必ず添え、ジャンル認知と拡散を同時に行う
- 拡散→プロフィール訪問→リンクタップ→報酬 の導線を構築
