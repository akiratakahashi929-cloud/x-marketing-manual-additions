## 🔍 差分分析の結果

WebマニュアルとUTAGE研修コンテンツを比較した差分分析の結果、大きく3つのカテゴリに分かれます：

### 1️⃣ Webのみに存在するコンテンツ（追加不要）

- Grokアルゴリズム解説・エンゲージメント重み付けの数値（Ch1）
- Viralダッシュボード活用全般（Ch6）
- PDCA改善サイクル・KPI基準値（Ch7）
- AIツール効率化のフレームワーク（Ch8）
- トラブル対応全般（Ch9）
- マインドセット理論（Ch11）

### 2️⃣ 宴のみに存在（今回の追加対象 ✅）

以下が宴から**プラスオンで追加すべき**コンテンツです：

| 追加先の章 | 追加コンテンツ | 概要 |
|-----------|-------------|------|
| **Ch3** | 7つの影響力借用戦略 | メンション・引用ポスト・通知オン・±500交流・拡散広告・バズリプ・トレンドワード |
| **Ch3** | 引用ポスト2つの型 | ①足し算型（具体例/体験/データ追加）②自己開示型（Before→After） |
| **Ch3** | 拡散広告の相場表 | ナノ〜マイクロインフルエンサーへの依頼費用と実績データ（8,000円→7万インプ） |
| **Ch3** | 拡散アカの見極め方 | ①引用RTリポスト率 ②リプ返答率の2基準 |
| **Ch3** | コンテンツ設計フレーム | 無料/有料の階層構造（何を出す/何を出さない） |
| **Ch3** | XTEPツール紹介 | ポスト最適化・自動DM/リプ・トラッキング |
| **Ch4** | 通知オン×即リアクション | 🔔設定方法・5分以内反応ルール |
| **Ch4** | ±500人交流戦略 | 選定基準3条件・交流5ステップ |
| **Ch4** | チーム内交流手順 | フォロー→通知→いいね→リプ→引用RTの5段階 |
| **Ch4** | 初期アカ日次/週次ルーティン | 初期アカ特化のタイムテーブル |
| **Ch4** | 成果の目安 | 1週/2週/1ヶ月のベンチマークと実績データ |
| **Ch4** | いいね基盤の形成 | 相互いいね→アカパワーUP |

### 3️⃣ 重複コンテンツ（Ch3のみ — 分析済み）

> **結論: 重複ではなく純粋な追加**
>
> 当初「引用ポストの足し算型/自己開示型」がWebのCh3と重複している可能性がありましたが、
> 精査の結果、**WebのCh3には引用ポストの記載は一切なし**（バズ投稿10テンプレートのみ）。
> よって、宴の引用ポスト戦略は**純粋な追加コンテンツ**として統合してOKです。

### Ch5「初期アカ成長戦略」について

`05_初期アカ成長戦略/参考_92Pスライド書き起こし.md` は、Ch3/Ch4に追加する内容の**元資料（92ページ分のスライド書き起こし全文）**です。

Ch3/Ch4の追加ファイルはこのスライドの内容を整理・構造化したものなので、**このファイル自体をWebに追加する必要はありません**。コンテンツの背景理解や詳細確認のための参照用として同梱しています。
